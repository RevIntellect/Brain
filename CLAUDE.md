# Brain

This project uses the [Superpowers](https://github.com/obra/superpowers) agentic skills framework.

## Superpowers Plugin

The superpowers plugin is installed and provides skills for:
- **Brainstorming** - Socratic design refinement before coding
- **Test-Driven Development** - RED-GREEN-REFACTOR discipline
- **Systematic Debugging** - 4-phase root cause analysis
- **Planning** - Detailed implementation plans with verification
- **Subagent-Driven Development** - Parallel task execution with review
- **Code Review** - Severity-based issue reporting
- **Git Worktrees** - Isolated development branches

Use skills via the `Skill` tool. Check for applicable skills before starting any task.
