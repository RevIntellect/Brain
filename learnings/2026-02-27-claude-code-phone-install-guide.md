# Installation Guide for Claude Code on Your Phone

**Date:** 2026-02-27
**Source:** [Claude Code Remote Control Video](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q)
**Tags:** claude-code, remote-control, mobile, installation, guide

---

This guide will help you set up and use the Claude Code Remote feature on your phone to access and control your Cloud Code sessions remotely.

---

## Step 1: Prepare Your Cloud Code Session on Your Laptop

- Open your Cloud Code terminal on your Mac (currently, the feature is available only for Mac and Pro users).
- In the terminal, type the command:

```
/remote-control
```

- This command activates the remote control mode for your current Cloud Code session. It may seem like nothing happens immediately, but it sets up the session for remote access from your phone. [(00:04:00)](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q&t=240)

---

## Step 2: Install the Native Claude App on Your Phone

- Download and install the native Claude app on your phone. This app allows you to connect directly to your Cloud Code sessions without needing VPN or remote desktop software. [(00:03:30)](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q&t=210)

---

## Step 3: Connect to Your Cloud Code Session from Your Phone

- Open the Claude app on your phone.
- Refresh or navigate to the Cloud Code section within the app.
- You should see your active Cloud Code session available to resume.
- Tap on the session to resume working on your Cloud Code terminal directly from your phone. [(00:04:30)](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q&t=270)

---

## Step 4: Starting a New Remote Session (Optional)

- You can also start a new remote session from scratch using the remote control option in the Claude app.
- Currently, starting a new session directly from the phone is a requested feature but not fully implemented; for now, you may need to start a session on your laptop first and then connect from your phone. [(00:05:00)](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q&t=300)

---

## Additional Notes

- The remote control feature allows you to use your laptop's or remote server's resources directly from your phone.
- No VPN or remote desktop is required; Anthropic handles the connection securely.
- Notifications and other features like thumbnails and screen clocks are available but may still be in development. [(00:06:00)](https://www.youtube.com/watch?v=ocQ7ZKhHU5Q&t=360)

---

By following these steps, you can seamlessly work on your Cloud Code projects from your phone, enabling remote productivity and flexibility.
