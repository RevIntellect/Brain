# Aaron Wolf — 30/60/90 Day Onboarding Dashboard

Interactive dashboard for tracking the 30-60-90 day onboarding plan to operationalize Gen AI for top-of-funnel pipeline and expert-level sales enablement.

## Features

- **Current Focus Mode** — Pin one task at a time to stay locked in; auto-advances when you complete it
- **Progress Tracking** — Interactive checkboxes on every task with overall and per-phase progress bars
- **Phase Navigation** — Filter by Phase 1 (Foundation), Phase 2 (Build), or Phase 3 (Scale)
- **Workstream Filtering** — Color-coded categories: Immersion, AI Build, Infrastructure, Pilot, Content
- **Stakeholder Mapping** — Key people attached to each task card
- **Personal Notes** — Jot down thoughts, blockers, or next steps on any task
- **Persistent State** — All progress, focus, and notes saved to localStorage automatically
- **Keyboard Shortcuts** — Press 1/2/3/0 to quickly switch phases
- **Mobile Responsive** — Tailwind CSS, works on any device

## Usage

Open `index.html` in any browser. No build step required.
